static make_safe_path (   $path,
  $fail = false,
  $createIfNotExists = true 
)
static

Check the path is inside the base folder or relative to base folder when safe paths are appended and the real path is resolved.

e.g. if config.allow_paths = [ '../logs' ] and web root is /var/sites/website/htdocs

then is_safe_path('/var/sites/website/logs') will return true but is_safe_path('/var/sites/website/conf') will return false

by default configuration the assets folder is always a safe path.

Parameters
string$path(no filename)
bool$failthrow an exception if test fails
bool$createIfNotExistscreate the directory if it doesn't exist (and is in assets folder only).
Returns
bool|string path or false if not safe
